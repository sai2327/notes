## 📌 What Is This Repository?

This is a **complete, structured, professional-grade cybersecurity learning repository** designed to take you from **absolute zero knowledge** to **advanced cybersecurity professional** in **12 months** of dedicated study.

This is NOT a collection of random links. This is a **full curriculum** with:
- Detailed explanations of every concept
- Hands-on labs and exercises
- Real-world projects
- Command-by-command walkthroughs
- Attack and defense methodologies
- Career preparation materials
- Certification study guides
- Interview preparation

## 📅 12-Month Learning Roadmap

### Overview

| Month | Focus Area | Difficulty | Hours/Week |
|-------|-----------|------------|------------|

| 4 | Web Development Fundamentals | Intermediate | 15-20 |
| 5 | Cybersecurity Fundamentals | Intermediate | 20 |
| 6 | Web Security + OWASP Top 10 | Intermediate | 20-25 |
| 7 | System Security + Privilege Escalation | Intermediate-Advanced | 20-25 |

### Month 1: Computer Fundamentals + Linux Basics

**Goal:** Understand how computers work and become comfortable with Linux.

#### Week 1-2: Computer Fundamentals
- How computers work (CPU, RAM, Storage)
- Operating system architecture
- Processes, threads, memory management
- File systems (NTFS, ext4)
- Boot process (BIOS/UEFI)
- Virtualization concepts

#### Week 3-4: Linux Basics
- Linux installation and setup
- File system navigation
- File permissions and ownership
- User and group management
- Basic bash commands
- Package management
- Process management

**Milestone:** Set up a Linux VM and navigate the system confidently.

---

### Month 2: Networking Fundamentals

**Goal:** Understand how networks work from physical layer to application layer.

#### Week 1: Network Foundations
- OSI Model (all 7 layers)
- TCP/IP Model
- IP addressing and subnetting

#### Week 2: Core Protocols
- TCP, UDP, ICMP
- DNS, DHCP, ARP
- HTTP/HTTPS

#### Week 3: Network Infrastructure
- Switches, Routers, Firewalls
- NAT, VPN, Proxy
- Routing concepts

#### Week 4: Packet Analysis
- Wireshark fundamentals
- Packet capture and analysis
- Network troubleshooting

**Milestone:** Capture and analyze network traffic, explain the TCP handshake.

---

### Month 3: Programming for Cybersecurity

**Goal:** Write Python scripts for security tasks and automate with Bash.

#### Week 1: Python Basics
- Variables, data types, operators
- Control flow, loops, functions
- File handling, error handling

#### Week 2: Python Intermediate
- Object-oriented programming
- Modules and libraries
- Regular expressions
- Socket programming

#### Week 3: Python Security Scripts
- Port scanner
- Password generator
- Hash cracker
- Network scanner

#### Week 4: Bash Scripting
- Bash fundamentals
- Automation scripts
- System administration scripts
- Security automation

**Milestone:** Build a working port scanner and network enumeration tool.

---

### Month 4: Web Development Fundamentals

**Goal:** Understand how web applications work to later find vulnerabilities.

#### Week 1: HTML & CSS
- HTML structure and elements
- Forms and input handling
- CSS basics

#### Week 2: JavaScript
- Variables, functions, DOM
- Event handling
- AJAX/Fetch API
- Client-side security

#### Week 3: Backend Basics
- HTTP methods (GET, POST, PUT, DELETE)
- Cookies, sessions, tokens
- SQL and databases
- Server-side logic

#### Week 4: Building Web Apps
- Login/authentication systems
- CRUD applications
- REST APIs
- Authentication flows

**Milestone:** Build a complete web application with authentication.

---

### Month 5: Cybersecurity Fundamentals

**Goal:** Master core security concepts and principles.

#### Week 1: Security Principles
- CIA Triad
- AAA (Authentication, Authorization, Accounting)
- Security models (Bell-LaPadula, Biba)
- Risk management

#### Week 2: Cryptography
- Symmetric encryption (AES, DES)
- Asymmetric encryption (RSA)
- Hashing (MD5, SHA)
- Digital signatures, PKI

#### Week 3: Network Security
- Firewalls, IDS/IPS
- VPN, SSL/TLS
- Network segmentation
- Zero Trust

#### Week 4: Threat Landscape
- Malware types
- Attack vectors
- Threat actors
- Kill Chain / MITRE ATT&CK

**Milestone:** Explain any security concept and implement basic crypto.

---

### Month 6: Web Security + OWASP Top 10

**Goal:** Find and exploit web vulnerabilities ethically.

#### Week 1: SQL Injection & XSS
- SQL Injection (Union, Blind, Error-based)
- Cross-Site Scripting (Reflected, Stored, DOM)
- Practice on DVWA/WebGoat

#### Week 2: Advanced Web Attacks
- CSRF, SSRF, XXE
- IDOR, File Upload vulnerabilities
- Command Injection

#### Week 3: Burp Suite Mastery
- Proxy, Repeater, Intruder
- Scanner, Decoder, Comparer
- Custom extensions

#### Week 4: Secure Coding & Defense
- Input validation
- Output encoding
- Parameterized queries
- Security headers

**Milestone:** Find and exploit all OWASP Top 10 vulnerabilities in labs.

---

### Month 7: System Security + Privilege Escalation

**Goal:** Escalate privileges on Linux and Windows systems.

#### Week 1: Linux Privilege Escalation
- SUID/SGID exploitation
- Cron job abuse
- Path injection
- Kernel exploits

#### Week 2: Windows Privilege Escalation
- Service misconfigurations
- Unquoted service paths
- DLL hijacking
- Token manipulation

#### Week 3: Windows Internals
- PowerShell for hackers
- Windows registry
- Active processes
- Windows services

#### Week 4: Defense & Hardening
- Linux hardening
- Windows hardening
- Audit policies
- Security baselines

**Milestone:** Root/Admin multiple practice machines through privilege escalation.

---

### Month 8: Penetration Testing Methodology

**Goal:** Perform complete penetration tests following professional methodology.

#### Week 1: Reconnaissance & OSINT
- Passive reconnaissance
- Active reconnaissance
- OSINT techniques
- Google dorking

#### Week 2: Scanning & Enumeration
- Nmap mastery
- Service enumeration
- Vulnerability scanning
- Banner grabbing

#### Week 3: Exploitation
- Metasploit Framework
- Manual exploitation
- Post-exploitation
- Pivoting

#### Week 4: Reporting & Professional Practice
- Report writing
- Executive summaries
- Technical findings
- Remediation recommendations

**Milestone:** Complete a full pentest on a practice network and write a professional report.

---

### Month 9: Active Directory Attacks

**Goal:** Attack and defend Active Directory environments.

#### Week 1: AD Fundamentals
- Domain Controllers
- Kerberos authentication
- LDAP, DNS in AD
- Group Policy

#### Week 2: AD Enumeration
- BloodHound
- PowerView
- LDAP queries
- Service enumeration

#### Week 3: AD Attacks
- Kerberoasting
- AS-REP Roasting
- Pass-the-Hash
- Pass-the-Ticket

#### Week 4: Lateral Movement & Persistence
- PsExec, WMI
- Golden/Silver tickets
- DCSync
- Persistence mechanisms

**Milestone:** Compromise an AD lab environment from initial access to domain admin.

---

### Month 10: Specializations

**Goal:** Deep-dive into your chosen specialization(s).

Choose 2-3 areas to focus on:

#### Track A: Reverse Engineering & Malware Analysis
- Assembly language
- Binary analysis with Ghidra
- Dynamic analysis with debuggers
- Malware behavior analysis

#### Track B: Cloud Security
- AWS/Azure security
- Container security (Docker/K8s)
- Cloud misconfigurations
- IAM exploitation

#### Track C: Blue Team / SOC
- SIEM (Splunk/ELK)
- Threat hunting
- Incident response
- Detection engineering

#### Track D: Red Team
- Adversary emulation
- C2 frameworks
- Evasion techniques
- OPSEC

**Milestone:** Complete specialization projects demonstrating expertise.

---

### Month 11: CTFs + Bug Bounty + Real Labs

**Goal:** Apply all knowledge in competitive and real-world scenarios.

#### Week 1-2: CTF Competitions
- Web challenges
- Pwn challenges
- Crypto challenges
- Forensics challenges

#### Week 3-4: Bug Bounty
- Methodology development
- Reconnaissance automation
- Vulnerability discovery
- Professional reporting

**Milestone:** Solve 50+ CTF challenges, submit first bug bounty reports.

---

### Month 12: Advanced Topics + Career

**Goal:** Prepare for professional cybersecurity career.

#### Week 1: Advanced Exploitation
- Exploit development basics
- Buffer overflows
- ROP chains
- Shellcoding

#### Week 2: Certification Preparation
- CompTIA Security+
- eJPT / PNPT
- OSCP preparation

#### Week 3: Career Preparation
- Resume building
- Portfolio optimization
- Interview preparation
- Networking strategies

#### Week 4: Final Projects
- Capstone project
- Professional portfolio
- Job applications
- Continued learning plan

**Milestone:** Ready to apply for junior cybersecurity positions.
